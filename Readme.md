# Price CLI php8.1
### Prerequisites
-  php8.1

## Setup

* In project directory:
  ```
  compose install

* Example Commands :
    * php index.php count_by_price_range 12.00 145.80
    * php index.php count_by_vendor_id 42 
* Run unitTests by composer :
    * composer tests
    * Or run :  vendor/bin/phpunit tests
